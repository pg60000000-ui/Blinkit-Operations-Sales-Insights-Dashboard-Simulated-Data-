# Blinkit-Operations-Sales-Insights-Dashboard-Simulated-Data-
Title
Blinkit Operations & Sales Insights Dashboard (Simulated Data)

Description
This PowerBi dashboard highlights key KPIs, sales trends, and outlet-level performance to quickly identify growth areas and underperforming segments. The insights support data-driven decisions around inventory planning, outlet expansion, and customer experience improvement.

Tech Stack
Power BI Desktop – Dashboard design & reporting
DAX – KPI calculations & measures
Power Query – Data cleaning & transformation
Microsoft Excel / CSV – Simulated dataset creation
Data Modeling – Relationships & schema design

Data
Type: Simulated / Mock data
Source: Self-generated for learning & portfolio purposes
Disclaimer: Not affiliated with Blinkit or its real business data

Features/Highlights:
Interactive KPI cards for sales, items sold, and customer ratings
Dynamic slicers for outlet type, size, and location
Drill-down capabilities for deeper performance analysis
Time-based trend analysis for sales insights
Category-wise and outlet-wise performance views
